# 🌦️ Real-Time Weather Dashboard

## 📌 Project Overview
This Power BI project provides a comprehensive look at current and forecasted weather conditions. By connecting directly to a Weather API, the report offers live updates on temperature, air quality (AQI), and precipitation probabilities.

## 🚀 Key Features
*   **Live API Integration:** Automated data refresh using Power Query to fetch real-time weather and pollution data.
*   **Air Quality Monitoring:** Visual representation of PM10, PM2.5, NO2, and CO levels with a status gauge.
*   **7-Day Forecast Trend:** A line chart visualizing temperature fluctuations over the coming week.
*   **Probability Tracking:** Horizontal bar charts indicating the daily 'Chance of Rain.'
*   **Dynamic UI:** Custom backgrounds and icons mapped to weather conditions for an immersive user experience.

## 🛠️ Skills & Tools Used
*   **Power Query:** Connected to REST API, parsed JSON data, and performed data cleaning.
*   **DAX:** Developed measures for calculating AQI status, average temperatures, and dynamic labels.
*   **Analytical Skills:** Identified key meteorological correlations between pressure, wind speed, and visibility.
*   **UI/UX Design:** Implemented a "Dark Mode" aesthetic with custom icon mapping.
